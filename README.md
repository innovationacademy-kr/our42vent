## Introduction

### ✅ 이벤트 일정 확인

- 캘린더 뷰에서 다양한 42 이벤트들의 일정을 확인할 수 있습니다.

### ✨ 이벤트 생성

- 특강, 외부 컨퍼런스, 개인 카뎃들이 여는 이벤트등을 캘린더 이벤트로 생성해서 42 서울 커뮤니티에 공유할 수 있습니다.

### 🔔 이벤트 구독

- 이벤트를 구독하여 나만의 캘린더로 관리할 수 있습니다.
- 알림설정을 통하여 브라우저 알림을 받을 수 있습니다.

### 📣 이벤트 홍보

- 이벤트 생성 후 자동으로 홍보글이 만들어져서 손쉽게 홍보할 수 있습니다.
